How to use

  • Use the buttons or click on the canvas to draw something
  • Click apply to apply the selected transformation to the current drawing
  • Click random to show the results of a "random walk" for the selected transformation
  • Click edit to create an editable copy of the selected transformation
  • Click copy to create an copy of the selected transformation
